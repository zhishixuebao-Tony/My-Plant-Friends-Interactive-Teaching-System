# My Plant Friends Interactive Teaching System (项目名称)

这是一个基于 [技术栈, 如 React + Node.js] 开发的植物互动教学系统。

## 🚀 功能特点
- 功能 1：植物识别...
- 功能 2：互动答题...

## 📦 目录结构
- `/frontend`: 前端代码 (React/Vue)
- `/backend`: 后端代码 (Express/Python)

## 🛠️ 如何在本地运行

### 1. 克隆项目
git clone https://github.com/zhishixuebao-Tony/My-Plant-Friends-Interactive-Teaching-System.git

### 2. 运行后端
cd backend
npm install
npm start

### 3. 运行前端
cd frontend
npm install
npm start
